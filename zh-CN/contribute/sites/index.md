## 使用 GitHub 网页编辑器

所有 SudoTV 创建的网站都是开源的。您可能会在每个网站页面看到一条页脚消息。它看起来像这样：

> 本站渲染自开源项目，{% github_edit_link "帮助本页面变得更好" %}。

点击链接来编辑当前显示的页面。

你可能需要一个 GitHub 账户来编辑一个页面或创建一个新页面。查看 [有关 GitHub 的主题](https://resource.sudo.tv/topic/github) 资源来了解更多关于 GitHub 的信息。
